# NSU Complain Lodge System
The purpose of NSUCLS is to create an online platform for the members of NSU to complain if they face any issue. NSUCLS can be used to lodge a complaint against NSU staff, faculty or students. The complaint lodged will then be instantly reviewed by a faculty or the system admin. The main aim of NSUCLS is to ensure a faster and easier approach for the members of NSU to complain and fix that issue. NSUCLS is also very user friendly for the helpers as it provides special features like lodging complaints by speech recognition.

## Features

* All users (Students, TA, RA, Lab Instructor, Faculty, Helper, Admin) can lodge complaints.
* ONLY Faculty or Admin can review complaints.
* Complain lodger can edit and update his/her complaint.
* The system admin can instantly create accounts, disable accounts, lodge complaint on behalf of any user.

## Operating Environment

* Platform: Web and Android
* Web Server: Any operation system with NodeJS RTE 16 or newer installed.
* Android App: Any system with a modern web browser or for android with android version 7.0+
* Database Configuration: MySQL
